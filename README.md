# Prodigy Infotech Data Science Internship

## Task 4: Sentiment Analysis and Visualization

In this task, I analyzed and visualized sentiment patterns in social media data to understand public opinion towards different entities.

## 1. Objective

To analyze customer sentiments (Positive, Negative, Neutral) and visualize their distribution to understand overall public opinion.

## 2. Data Cleaning

* Removed irrelevant sentiment values
* Filtered only valid categories: Positive, Negative, Neutral
* Verified dataset structure

## 3. Data Analysis

* Calculated count of each sentiment using `value_counts()`
* Identified distribution of Positive, Negative, and Neutral sentiments

## 4. Data Visualization

### 4.1 Bar Chart

* Displayed count of each sentiment category
* Helped compare the number of responses

### 4.2 Pie Chart

* Showed percentage distribution of sentiments
* Provided an overall view of public opinion

## 5. Tools and Technologies

* Python
* Pandas
* Matplotlib

## 6. Key Learnings

* Importance of data cleaning before analysis
* Understanding categorical data distribution
* Using value_counts() for quick insights
* Visualizing data using bar and pie charts

## 8. Conclusion

The analysis shows that negative sentiments are slightly higher than positive ones, while neutral sentiments also form a significant portion. This indicates a balanced but slightly negative public opinion overall.
